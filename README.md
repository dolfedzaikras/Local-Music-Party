Local-Music-Party 
**Project by 4-MD-25 Dolmatsyn Artem, Fedorova Renata, Krasnokutskaya Elizaveta, Zaicev Danil**

 
Контекст: Компания “Local music party” разработало мобильное приложение для поиска локальных
мероприятий.
󾠮 
Разработать логотип компании в векторном формате. Пожелания заказчика: “Мы хотим
использовать яркие цвета, такие как фиолетовые и коралловый. Логотип видим в комбинированном
стиле с яркой типографикой и простым визуальным знаком.”
󾠯 
Разработать дизайн одностраничного лендинг-сайта. Рекомендуется использовать инструмент
Figma. Выравнивать элементы сайта по модульной сетке. Пожелания заказчика: современный
дизайн в темной теме, наличие мокапов.
󾠰 
Сверстать сайт на языках html, css, js. Рекомендуется использовать фраемворк bootstrap.
󾠱 
Добавить анимации элементам сайта через js фраемворки.
󾠲 
Подключить верстку сайта к базе данных.
